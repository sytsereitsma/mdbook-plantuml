# Windows MSI installer creation

## Prerequisites

[WiX Toolset](https://github.com/wixtoolset/wix/releases). Download and install the cli (command line) version.
[dotnet](https://dotnet.microsoft.com/en-us/download).

## Building the MSI installer
1. Open a powershell terminal and navigate to the `scripts/msi_installer` directory.
2. Run `create_installer.ps1`
